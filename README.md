# 🎓 JS Ideal School Portal

A modern and user-friendly School Management System designed to simplify academic and administrative operations. The platform provides efficient management of students, teachers, classes, attendance, and academic records through a centralized digital solution.

## 🚀 Features

- 👨‍🎓 Student Management
  - Student registration
  - Student profile management
  - Academic record tracking

- 👨‍🏫 Teacher Management
  - Teacher information management
  - Class assignment tracking

- 📚 Class Management
  - Course and class organization
  - Schedule management

- 📅 Attendance System
  - Student attendance tracking
  - Attendance reporting

- 📊 Results & Academic Records
  - Result management
  - Performance monitoring

- 🔐 Secure Authentication
  - User login system
  - Role-based access control

- 📱 Responsive Design
  - Desktop-friendly interface
  - Mobile-responsive experience

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/PARZIVAL283/js-ideal-school-portal.git
```

2. Move the project folder to your XAMPP `htdocs` directory.

3. Start:
   - Apache
   - MySQL

4. Import the database:
   - Open phpMyAdmin
   - Create a new database
   - Import the provided SQL file

5. Open your browser:

```text
http://localhost/js-ideal-school-portal
```
